# Todo-List Application

## Description
A simple and interactive web-based task management application built with JavaScript.

## Features
✅ Add new tasks
✅ Delete tasks
✅ Mark tasks as complete
✅ Responsive design for all devices

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Use
1. Open the `index.html` file in your browser
2. Type your task in the input field
3. Click "Add" to add a new task
4. Click on a task to mark it complete
5. Click delete button to remove a task

## Installation
```bash
git clone https://github.com/AtousaYekrang/Todo-List.git
cd Todo-List
open index.html
```
